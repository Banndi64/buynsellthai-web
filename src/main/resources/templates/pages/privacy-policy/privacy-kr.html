데이터 수집의 법적 근거 및 목적

당사는 개인정보 보호법 B.E. 2562에 따라 법적 근거를 바탕으로 사용자의 데이터를 수집하며, 이를 안전하게 보호합니다.

수집하는 데이터

  • GPS 데이터: 실시간 위치 정보
  • Wi-Fi 연결 정보: 네트워크 SSID, 신호 강도
  • 기지국 데이터: 네트워크 기반 위치 정보
  • 블루투스 비콘 데이터: 근접 기반 서비스 정보
  • IP 주소: 보안 및 위치 확인
  • 사용자 입력 데이터: 사용자가 직접 입력한 위치 정보

데이터 수집 방법

당사는 GPS, Wi-Fi, 블루투스 및 사용자 입력을 통해 데이터를 수집합니다.

사용자 동의 및 알림

사용자는 데이터를 제공하기 전 명시적 동의를 해야 하며, 언제든지 동의를 철회할 수 있습니다.

데이터 보안 및 보호

  • 암호화: 데이터는 저장 및 전송 시 암호화됩니다.
  • 액세스 제어: 승인된 사용자만 데이터에 접근 가능합니다.
  • 보안 감사: 정기적인 보안 감사를 실시합니다.

데이터 저장 및 삭제

데이터는 필요 기간 동안만 보관되며, 이후 안전하게 삭제됩니다.

사용자의 권리

  • 액세스 권한: 사용자는 자신의 데이터에 접근할 권리가 있습니다.
  • 수정 요청: 부정확한 데이터를 수정할 수 있습니다.
  • 동의 철회: 사용자는 데이터 제공 동의를 철회할 수 있습니다.
  • 데이터 삭제 요청: 사용자는 데이터를 삭제하도록 요청할 수 있습니다.

법률 준수
