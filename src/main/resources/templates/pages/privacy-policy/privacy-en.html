Privacy Policy

Legal Basis and Objectives of Data Collection

The collection of data is governed by the Personal Data Protection Act B.E. 2562, ensuring transparency and security in handling user information.

Collected Data

  • GPS Data: Real-time location coordinates for personalized services.
  • Wi-Fi Access Information: SSID and signal strength for indoor positioning.
  • Cell Tower Data: Used for network-based positioning.
  • Bluetooth Beacon Data: Proximity-based services.
  • IP Address: For security and approximate location identification.
  • User-entered Data: Manually entered location data for service customization.

Data Collection Methods

Data is collected through GPS tracking, Wi-Fi scanning, event-based triggers, and user input.

User Consent and Notifications

Users provide explicit consent for data collection, with options to withdraw consent at any time.

Data Security and Protection

  • Encryption: Secures data during storage and transmission.
  • Access Control: Restricted access to authorized personnel only.
  • Regular Audits: Ensuring compliance with data protection laws.

Storage and Disposal of Location Data

Data is retained only for the necessary duration and securely disposed of once the purpose is fulfilled.

User Rights

  • Right to Access: Users can request their data details.
  • Right to Rectification: Users can correct inaccurate data.
  • Right to Withdraw Consent: Users can revoke data collection consent.
  • Right to Erase Data: Users can request data deletion.

Legal Compliance

The policy complies with the Personal Data Protection Act B.E. 2562 and other applicable Thai laws.
